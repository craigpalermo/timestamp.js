Overview

Timestamp.js is a simple library for converting written date strings or JavaScript Date objects to various timestamp formats.

Warning! Timezones are not currently supported in input strings. All outputs will be relative to your current timezone as detected by JavaScript's Date class. Timezone support will be added in a later version.

Supported Input Formats

Timestamp supports several input formats for dates and times, but both date AND time are required, separated on or more spaces.

Example: 

"01/2/2015 04:15pm"

Dates

                      1/2/2015
                      01/2/2015
                      1-2-2015
                      01-2-2015

Times

                      4:15pm
                      04:15pm
                      16:15
